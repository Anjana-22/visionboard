# Digital Vision Board

A simple vision board website where you can add goals, images, checklists, reminders, and random life plans you swear you’ll complete this year.

Built because normal notes apps are boring.

## Demo

click here - visionboard-rouge.vercel.app

## Features

* Add goals and notes
* Upload inspiration images
* Create checklists
* Add deadlines
* Pin important cards
* Dark mode
* Search and filters
* Saves data using local storage

## Tech stack

* HTML
* CSS
* JavaScript

## Future Scope

Possible upgrades and features planned for the future:
